# Bilimsel Hesap Makinesi | Scientific Calculator 🧮
Bu proje C++ dilinde yazılmış bilimsel bir hesap makinesine aittir.

<br>

*This project belongs to a scientific calculator written in C++.*

<br>

## ⚙️ Özellikler | Features

### ➕ Temel İşlem | Basic Operations:
- 🇹🇷 **TR**: Toplama (+), Çıkarma (-), Çarpma (*), Bölme (/) işlemlerini yapabilme.
- 🇬🇧 **EN**: Ability to perform Addition (+), Subtraction (-), Multiplication (*), and Division (/) operations.

<br>

### 📐 Trigonometri | Trigonometry:

- 🇹🇷 **TR**: Sinüs (sin), Kosinüs (cos), Tanjant (tan), Kotanjant (cot) değerlerini hesaplayabilme.

- 🇬🇧 **EN**: Ability to calculate Sine (sin), Cosine (cos), Tangent (tan), and Cotangent (cot) values.

<br>

### 📈 Logaritma | Logarithm:

- 🇹🇷 **TR**: Onluk tabanda (log10) logaritma hesaplama özelliği.

- 🇬🇧 **EN**: Base-10 logarithm calculation feature.
